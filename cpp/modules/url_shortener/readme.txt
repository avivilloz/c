Description:
simple url shortener, with only the representation of the algorithm, some sample classes and the test.

1- type "make" to compile.
2- type "make run" to run the test.
3- type "make clean" to clean the executable and compile again.
4- if desired, opening the makefile lets you change some of the arguments, like default (raw) url and path to storage.

*the test is very simple and it will print only 5 urls stored and retrived by the application.
